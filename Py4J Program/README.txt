Windows Instructions:

1: (Optional) Populate "transactions.txt" with data using this format:
	Account#, Transaction#, Amount, Date, Currency, Conversion_Rate, Description
2: Run "make.bat" to compile and run the Java gateway server. Leave this window alone.
3: Run "run.bat" to run the Python script.
4: (Optional) Run "clean.bat" to delete the .class files generated by the Java compilation.
